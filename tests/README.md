# Tests

In order to run the tests you will need to insert your AWS temporary credentials to **config.yaml** s3client:

- aws_access_key_id
- aws_secret_access_key
- aws_session_token

Follow the instructions [here](https://logzio.atlassian.net/wiki/spaces/DEV/pages/1068335431/Generating+AWS+temporary+credentials) to generate your AWS temporary credentials.
